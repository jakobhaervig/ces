    Abstract

    Note the abstract submitted on {{site.data.dates.abstract_submission_deadline}} should be written in Markdown format (extension .md). A markdown file is simply a plain text file. When submitting your abstract please make sure to name the file as your group email [group-number].md, e.g. et-24-tepe-1-700.md. If in doubt, reach out to ces-conference@energy.energy.aau.aau.dk with questions. More specific guidelines are written in the template.
